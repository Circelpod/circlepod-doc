:::caution Please check these things below:

## What is "Vote for podcast"?
**Vote for podcast** is a way to generate the channel pool. There are so many channel pools in `Channel`. 
If you want to support your loved podcast on Channel, you need to **Vote for your loved podcast** first.
On devnet, the threshold of generating a channel pool is 30 votes.
