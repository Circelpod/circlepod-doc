## 什麼是頻道成員

頻道成員是頻道池的操作賬號，一個頻道對應一個頻道會員（類似一個銀行中的帳號概念）。

在能夠進入頻道池之前，必須在頻道中創建一個會員帳戶。

每個成員都有四種類型的 Token 保險庫，組成了代表成員擁有的一組餘額：

* 金庫（Value）：可用餘額零利息賺取代幣賬戶，沒有任何限制。
* 待提領（Vault Pending Withdrawal Balance）：取消支持（UnSUP）的資金會導致解綁時間鎖定。（7 天）
* 質押（Stake）：支持（SUP）節目，質押的代幣總量。
* 質押權益（Stake pool token）：通過質押創建的池代幣總量`（質押權益=質押權益代幣數量*質押權益代幣價格）`。
