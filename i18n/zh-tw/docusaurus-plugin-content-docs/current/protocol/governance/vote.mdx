# 為您喜愛的 Podcast 投票

:::caution 請檢查以下事項：

- 您的 Circlepod 錢包已連接！更多詳情：[點擊](/docs/wallet/connect-wallet)
- 您有足夠的 CPX 投票！更多詳情：[點擊](/docs/wallet/airdrop)
:::

### 第 1 步. 選擇您喜愛的 Podcast

![申請 CPX 空投](/img/event/vote/choose-to-vote.png)

### 第 2 步. 輸入您要投票的票數

如果 Podcast有 30 票，就會生成頻道池。之後，您可以支持該 Podcast！
![申請 CPX 空投](/img/event/vote/ticket.png)
:::caution

- 您只能為每個 Podcast投票一次。
:::

### 第 3 步. Podcast 投票成功！

![申請 CPX 空投](/img/event/vote/vote-successfully.png)

:::tip
- 總票數不是即時更新的。我們每 30 分鐘更新一次總投票數！
- 測試網站的投票門檻為30票，正式上線前會公佈正式版本的門檻
:::

  👉 [YouTube](https://www.youtube.com/watch?v=AtLtVVqO7Gc) 上有更多詳情！ 😍
