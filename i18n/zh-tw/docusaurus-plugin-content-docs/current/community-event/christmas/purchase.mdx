# 購買方案

## NFT
購買 Circlepod 商城內，，滿額的商品可自選不同數量的 [NFT](/docs/community-event/christmas/nft) 款式（共 3 款）

- 一片葉子 🌱： 金額達 $59 ，可自選 1 款 NFT。
- 一堆葉子 🌱🌱: 金額達 $108 ，可自選 2 款 NFT。
- 一袋葉子 🌱🌱🌱： 金額達 $159 ，可自選 3 款 NFT。

:::caution
❗️上述金額為不含運費的總金額❗️
:::

## 特殊角色 - Circlepod Shoper
❗️只要符合以上任何一個方案，就可以在 Discord 中獲得**特殊角色**❗️

- 我們將優先提供商城的優惠資訊給此群組。
- 可參與討論實體產品製作。


## 邀請碼
只要符合以上任一方案，即可獲得 App 的**邀請碼**，並解鎖 Circlepod App 的核心功能。
