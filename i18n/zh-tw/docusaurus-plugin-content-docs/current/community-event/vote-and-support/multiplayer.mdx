# 多人任務
您已經完成了單人任務嗎？ 這裡有一個好消息要告訴您。
如果更多人完成單人任務，我們將解鎖更多空投。 有以下規則：

- [x] 當 1000 人完成單人任務時，我們將解鎖 50 張門票。
- [x] 當 1500 人完成單人任務時，我們將解鎖 150 張門票。
- [x] 當 2500 人完成單人任務，解鎖 240 張門票。
- [x] 當 3500 人完成單人任務時，我們將解鎖 350 張門票。
- [x] 當 5000 人完成單人任務時，我們將解鎖 510 張門票。

- [x] 當 10,000 人完成單人任務後，我們將解鎖 1,150 張門票。
- [x] 當 13,000 人完成單人任務時，我們將解鎖 1,500 張門票。
- [x] 當 15,000 人完成單人任務時，我們將解鎖 1,750 張門票。
