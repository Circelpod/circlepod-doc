2. 若未集滿 5 張票，仍有機會獲得空投，但獲得的數量與機率相較於擁有 5 張票的人少！
3. 擁有前 5 名票數的獨家優惠：
    1. 第一名 🥇 可獲得 1000 CPX 
    2. 第二名 🥈 可獲得 700 CPX
    3. 第三名 🥉 可獲得 500 CPX
    4. 第四名 🏅 可獲得 250 CPX
    5. 第五名 🎖 可獲得 100 CPX
