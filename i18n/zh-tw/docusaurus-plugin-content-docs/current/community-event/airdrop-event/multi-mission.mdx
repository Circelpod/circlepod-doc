#### New Available <Highlight color="#25c2a0">Devnet</Highlight>


**多人任務** 仍然適用，解鎖的票數會回饋到擁有至少 1 張空投票的人身上。因此，愈多人參與，增加的票數就會更多！

![img](/img/event/tickets.png)


### 多人任務
您已經完成個人任務了嗎？ 這裡有一個好消息要告訴您。 👏
如果有更多人完成個人任務，我們將解鎖更多空投。 以下為解鎖規則：

- [x] 當 1000 人完成個人任務時，我們將解鎖 50 張票。
- [x] 當 1500 人完成個人任務時，我們將解鎖 150 張票。
- [x] 當 2500 人完成個人任務時，我們將解鎖 240 張票。
- [x] 當 3500 人完成個人任務時，我們將解鎖 350 張票。
- [ ] 當 5000 人完成個人任務時，我們將解鎖 510 張票。
- [ ] 當 7000 人完成個人任務時，我們將解鎖 750 張票。
- [ ] 當 10000 人完成個人任務時，我們將解鎖 1150 張票。
- [ ] 當 13000 人完成個人任務時，我們將解鎖 1500 張票。
- [ ] 當 15000 人完成個人任務時，我們將解鎖 1750 張票。

快將此活動分享給您的朋友！ 🔥

例如：
如果有 24000 張活動票券，5000 人參加活動。 **票池**將有 24000 + 510 = 24510 張票。

:::tip
如果完成個人任務的人數超出這個範圍，
Circlepod 官方將在未來公布解鎖票券總數！
您可以在**儀表板**上看到進度
![img](/img/event/stake.png)
:::
