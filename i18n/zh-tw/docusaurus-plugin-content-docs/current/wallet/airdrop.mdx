# 如何在 devnet 上獲取 CPX？

:::caution 請檢查以下事項：

- 您的 Circlepod 錢包已連接！更多詳情：[點擊](/docs/wallet/connect-wallet)
- 您的推特帳戶已連接！更多詳情：[點擊](/docs/wallet/connect-twitter)
:::

### 第 1 步. 點擊`Apply CPX Airdrops`在 devnet 上獲得 10 個 CPX。

![申請CPX空投](/img/event/airdrop/apply-airdrop.png)

### 第 2 步. 成功在 devnet 上獲得 10 個 CPX！

![申請CPX空投](/img/event/airdrop/get-cpx-successfully.png)

:::caution 為什麼 CPX 還是 0 ？
CPX 有 2 種狀態！ `Created` 及 `Dropped`。請先確認 CPX 狀態顯示為 dropped，您才會得到 10 CPX！
![cpx](/img/wallet/droped-cpx.png)
:::
