# 关于 Circlepod

Circlepod 会为每个 Podcaster 制作节目 NFT，利用流动性挖矿的方式开放听众与粉丝进行质押（声援），Podcaster 与质押者都可以得到 Token 收益。除此之外，Circlepod 也正在规划生态系互动方式。比如，Podcaster 与听众对于整体生态进行贡献时，也可以获得 Token 收益。 Circlepod 也会帮助 Podcaster  制作单集版权 NFT，协助 Podcaster 透过 NFT 与智能合约完成自动化的付费内容与版税收入。所有收听数据与热点资料也会开放区块链使用，使得数位内容创作拥有一个数据平台。

```
🚀 使命：鼓励与加速全球数位内容创作领域的变革与发展
👍 愿景：让每一份用心的创作内容都可以得到应有的价值
```
