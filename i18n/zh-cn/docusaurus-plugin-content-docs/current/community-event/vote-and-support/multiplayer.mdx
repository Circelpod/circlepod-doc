# 多人任务
您已经完成了单人任务吗？这里有一个好消息要告诉您。
如果更多人完成单人任务，我们将解锁更多空投。有以下规则：

- [x] 当 1000 人完成单人任务时，我们将解锁 50 张门票。
- [x] 当 1500 人完成单人任务时，我们将解锁 150 张门票。
- [x] 当 2500 人完成单人任务，解锁 240 张门票。
- [x] 当 3500 人完成单人任务时，我们将解锁 350 张门票。
- [x] 当 5000 人完成单人任务时，我们将解锁 510 张门票。
- [x] 当 7000 人完成单人任务时，我们将解锁 750 张门票。
- [x] 当 10,000 人完成单人任务后，我们将解锁 1,150 张门票。
- [x] 当 13,000 人完成单人任务时，我们将解锁 1,500 张门票。
- [x] 当 15,000 人完成单人任务时，我们将解锁 1,750 张门票。

将此活动分享给您的朋友！

:::tip
如果超出这个范围的人完成单人任务，
Circlepod 官方将在未来公布解锁门票总数！
:::
