# 购买方案

## NFT
购买 Circlepod 商城内，，满额的商品可自选不同数量的 [NFT](/docs/community-event/christmas/nft) 款式（共 3 款）

- 一片叶子 🌱： 金额达 $59 ，可自选 1 款 NFT。
- 一堆叶子 🌱🌱: 金额达 $108 ，可自选 2 款 NFT。
- 一袋叶子 🌱🌱🌱： 金额达 $159 ，可自选 3 款 NFT。

:::caution
❗️上述金额为不含运费的总金额❗️
:::

## 特殊角色 - Circlepod Shoper
❗️只要符合以上任何一个方案，就可以在 Discord 中获得**特殊角色**❗️

- 我们将优先提供商城的优惠资讯给此群组。
- 可参与讨论实体产品制作。


## 邀请码
只要符合以上任一方案，即可获得 App 的**邀请码**，并解锁 Circlepod App 的核心功能。
