**多人任务** 仍然适用，解锁的票数会回馈到拥有至少 1 张空投票的人身上。因此，愈多人参与，增加的票数就会更多！

![img](/img/event/tickets.png)


### 多人任务
您已经完成单人任务了吗？这里有一个好消息要告诉您。 👏
如果有更多人完成单人任务，我们将解锁更多空投。以下为解锁规则：

- [x] 当 1000 人完成单人任务时，我们将解锁 50 张门票。
- [x] 当 1500 人完成单人任务时，我们将解锁 150 张门票。
- [x] 当 2500 人完成单人任务时，我们将解锁 240 张门票。
- [x] 当 3500 人完成单人任务时，我们将解锁 350 张门票。
- [x] 当 5000 人完成单人任务时，我们将解锁 510 张门票。
- [x] 当 7000 人完成单人任务时，我们将解锁 750 张门票。
- [x] 当 10000 人完成单人任务时，我们将解锁 1150 张门票。
- [x] 当 13000 人完成单人任务时，我们将解锁 1500 张门票。
- [x] 当 15000 人完成单人任务时，我们将解锁 1750 张门票。

将此活动分享给您的朋友！ 🔥

例如：
如果有 24000 张活动票券，5000 人参加活动。**票池**将有 24000 + 510 = 24510 张票。

:::tip
如果完成单人任务的人数超出这个范围，
Circlepod 官方将在未来公布解锁门票总数！
您可以在**仪表板**上看到进度
![img](/img/event/stake.png)
:::
