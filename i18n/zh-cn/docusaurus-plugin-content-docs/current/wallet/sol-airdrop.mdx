# 如何在 devnet 上获取 SOL？

您有 2 种方法可以在 devnet 上获取 SOL！
如果您没有 CPX 钱包，请参考第一种方式`通过 Circlepod 的 Discord bot 获取 CPX`

## 通过 Circlepod 的 Discord 机器人获取 CPX

### 第 1 步. 加入 [Discord](https://discord.gg/D2pNA2e6Dw)！

### 第 2 步. 进入 **App** 下的 #devnet-sol-airdrop 频道

![SOL空投](/img/wallet/discord.png)

### 第 3 步. 输入 `!airdrop '您的 Solana 地址'`

![SOL 空投](/img/wallet/enter-address.png)

### 第 4 步. 完成

![SOL空投](/img/wallet/sol-airdrop.png)

您会收到这样的消息：

```
申请空投 1 个 SOL

签名：4bonnLvNw8FWpKj5Bxv3FSE8FoWpZi3uxxicHhpxfuRBBuxrD3ZYSJaEesEXCcz8S6NX6SWdJJFNuvsHNVfyRy8N

2.96896284 SOL
```
:::note
现在我们每次提供 0.1 SOL！
:::

## 通过 Circlepod Protocol 网站获取

如果您已经拥有 CPX 帐户，您可以按照以下步骤操作。

:::caution 请检查以下事项：

- 您的 Circlepod 钱包已连接！更多详情：[点击](/docs/wallet/connect-wallet)
- 您的堆特帐户已连接！更多详情：[点击](/docs/wallet/connect-twitter)
:::

### 第 1 步. 点击`Apply Solana Airdrops`在 devnet 上获得 0.1 SOL。

![申请SOL空投](/img/wallet/get-sol.png)

### 第 2 步. 成功在 devnet 上获取 0.1 SOL！

![申请SOL空投](/img/wallet/sol-succesfully.png)
