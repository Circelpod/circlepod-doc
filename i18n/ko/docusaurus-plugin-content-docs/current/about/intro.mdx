# 關於 Circlepod

Circlepod 會為每個 Podcaster 製作節目 NFT，利用流動性挖礦的方式開放聽眾與粉絲進行質押（聲援），Podcaster 與質押者都可以得到 Token 收益。除此之外，Circlepod 也正在規劃生態系互動方式。比如，Podcaster 與聽眾對於整體生態進行貢獻時，也可以獲得 Token 收益。Circlepod 也會幫助 Podcaster 製作單集版權 NFT，協助 Podcaster 透過 NFT 與智能合約完成自動化的付費內容與版稅收入。所有收聽數據與熱點資料也會開放區塊鏈使用，使得數位內容創作擁有一個數據平台。

```
🚀 使命：鼓勵與加速全球數位內容創作領域的變革與發展
👍 願景：讓每一份用心的創作內容都可以得到應有的價值
```
